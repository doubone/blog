# IE 调试
## 背景说明  
chrome本地正常调试的web项目，在IE浏览器中输入调试地址无响应  
## 解决思路 

配置IE浏览器的局域网配置脚本地址
1. 进入internet选项
2. 点击连接选项卡，打开局域网设置
3. 勾选使用自动配置脚本选项，输入web项目调试地址，确定
4. 保存配置项，重新输入调试地址
