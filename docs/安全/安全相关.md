
# WEB系统安全
1. 登录认证防暴力破解  
登录认证模块提供防暴力破解机制
2. 验证码或者多次连续登录失败后锁定账号
登录5次失败后，账号锁定15分钟  
3. 对于每一个需要授权访问的页面都必须核实用户的访问标识是否允许。
用户是否被授权去某个页面操作，防止URL越权
4. 登录口令保护  
用户登录后保存sessionId在localstoage或者cookie中，再去请求接口时需要加密登录口令后再传输，不能直接拿原始sessionId请求，访问api越权操作
