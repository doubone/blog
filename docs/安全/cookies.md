# HTTP Cookies
HTTP Cookie（也叫 Web Cookie 或浏览器 Cookie）是服务器发送到用户浏览器并保存在本地的一小块数据，它会在浏览器下次向同一服务器再发起请求时被携带并发送到服务器上。Cookie 使基于无状态的HTTP协议记录稳定的状态信息成为了可能。  

Cookies特性  
* 前端数据存储
* 后端通过http头设置
* 请求时通过http头传给后端
* 前端可读写
* 遵守同源策略 (协议、域名、端口)·
